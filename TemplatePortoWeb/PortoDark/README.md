# PortoDark
Dark-themed portfolio with parallax effect.

## Features
- Elegant dark mode.
- Responsive design.
- Parallax decor.

## Customization
- `index.html`: Update name, tagline, links.
- `style.css`: Adjust colors, parallax speed.
- `profile.jpg` & `decor.png`: Replace images.

## Usage
Open `index.html` in a browser.
