# PortoVivid
Bright-themed portfolio with bounce animation.

## Features
- Vivid color palette.
- Responsive design.
- Bounce effect.

## Customization
- `index.html`: Update name, tagline, links.
- `style.css`: Adjust colors, bounce height.
- `profile.jpg` & `decor.png`: Replace images.

## Usage
Open `index.html` in a browser.
