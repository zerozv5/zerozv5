# PortoTech
Tech-themed portfolio with card flip effect.

## Features
- 3D flip animation.
- Responsive design.
- Futuristic aesthetic.

## Customization
- `index.html`: Update name, tagline, links.
- `style.css`: Adjust colors, flip speed.
- `profile.jpg` & `decor.png`: Replace images.

## Usage
Open `index.html` in a browser.
