# PortoRetro
Retro 80s-themed portfolio with pixel effects.

## Features
- Pixelated design.
- Responsive layout.
- Blink animation.

## Customization
- `index.html`: Update name, tagline, links.
- `style.css`: Adjust colors, sizes.
- `profile.jpg` & `decor.png`: Replace images.

## Usage
Open `index.html` in a browser.
