# PortoMinimal
Minimalist portfolio with scroll reveal.

## Features
- Clean design.
- Responsive layout.
- Scroll fade-in.

## Customization
- `index.html`: Update name, tagline, links.
- `style.css`: Adjust colors, sizes.
- `profile.jpg` & `decor.png`: Replace images.

## Usage
Open `index.html` in a browser.
